# Airbyte

INGESTION

Airbyte is an open-source data integration platform that enables the seamless transfer of data from various sources to data warehouses, lakes, and other destinations. It supports a vast range of connectors and provides an intuitive interface for managing data pipelines, ensuring data consistency and reliability. Airbyte offers both a community edition and a cloud service, allowing flexibility in deployment and scalability.

## Airbyte + MotherDuck

Airbyte integrates with MotherDuck by facilitating the extraction, transformation, and loading (ETL) of data from numerous sources directly into the MotherDuck cloud data warehouse. Utilizing Airbyte's extensive library of connectors, users can automate and streamline the process of moving data into MotherDuck, ensuring that their cloud data warehouse remains up-to-date and easily accessible for analysis.
