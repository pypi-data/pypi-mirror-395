# Fivetran

INGESTION

Fivetran is a data integration tool that automates the process of extracting, loading, and transforming data from various sources into your data warehouse. It offers pre-built connectors for a wide range of data sources, ensuring reliable, consistent, and up-to-date data for analysis. Fivetran focuses on ease of use, requiring minimal maintenance while providing robust data pipeline solutions.

## Fivetran + MotherDuck

Fivetran integrates with MotherDuck by seamlessly connecting to it as a destination for data ingestion. Users can set up Fivetran to automatically extract data from various sources and load it directly into MotherDuck, leveraging MotherDuck's high-performance analytical capabilities. This integration simplifies the data pipeline setup and ensures that data is consistently and accurately delivered to MotherDuck for analysis.
