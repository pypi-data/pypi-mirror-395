# Preset

BUSINESS INTELLIGENCE

Preset is a modern data exploration and visualization platform built on Apache Superset. It enables users to create, share, and collaborate on interactive dashboards and reports with ease. With robust support for SQL-based querying and a variety of visualization options, Preset is designed to empower data analysts and scientists to gain insights quickly.

## Preset + MotherDuck

Preset integrates with MotherDuck by connecting to it as a SQL-based data source. Users can easily query data stored in MotherDuck's cloud data warehouse using Preset's interface, enabling seamless data visualization and exploration. This integration leverages MotherDuck's efficient, high-performance querying capabilities to enhance the overall data analysis experience.
