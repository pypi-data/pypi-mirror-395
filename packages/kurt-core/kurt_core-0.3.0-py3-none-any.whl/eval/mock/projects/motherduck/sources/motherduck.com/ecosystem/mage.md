# Mage

ORCHESTRATION

INGESTION

Mage is an open-source data pipeline tool designed to simplify the process of building, running, and managing data pipelines. It offers a user-friendly interface, supports scheduling and monitoring, and integrates seamlessly with various data sources and destinations. Mage provides built-in features for data transformations, ensuring efficient and reliable data workflows. They call themselves the modern replacement for Airflow.

## Mage + MotherDuck

Mage integrates with MotherDuck by allowing users to create data pipelines that connect to MotherDuck's cloud data warehouse. This integration enables seamless data ingestion, transformation, and analysis within MotherDuck, leveraging DuckDB's high-performance query engine for efficient data processing.
