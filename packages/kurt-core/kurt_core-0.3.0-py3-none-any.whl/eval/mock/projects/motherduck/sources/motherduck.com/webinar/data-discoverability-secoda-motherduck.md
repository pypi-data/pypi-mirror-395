## Take flight with MotherDuck and Secoda

Join Secoda and MotherDuck for a masterclass in using dbt, MotherDuck, and Secoda to enable data producers and consumers, regardless of technical ability, to easily locate and access the data they need.

You'll hear from [Andrew McEwen](https://x.com/andrew_mcewen), Co-founder and CTO at Secoda, and [Alex Monahan](https://x.com/__AlexMonahan__), Forward Deployed Software Engineer at MotherDuck.
