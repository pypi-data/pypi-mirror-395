# Cube

BUSINESS INTELLIGENCE

Cube is an API-first analytics platform that enables data engineers and developers to build data applications. It provides a semantic layer to unify data sources, caching, and pre-aggregation for fast query performance. Cube supports various data sources and visualization tools, making it a versatile solution for creating and managing analytical data workflows.

## Cube + MotherDuck

Cube integrates with MotherDuck by leveraging its DuckDB-based architecture to efficiently query and analyze data stored in the MotherDuck cloud data warehouse. This integration enables Cube to utilize MotherDuck's high-performance SQL engine, providing fast and seamless access to large datasets for building robust data applications.
