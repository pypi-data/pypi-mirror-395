# Hex

BUSINESS INTELLIGENCE

Hex is a collaborative data workspace that allows data scientists, analysts, and engineers to create, share, and run complex data workflows. It combines powerful data notebooks with a rich set of visualization tools and integrations, enabling seamless collaboration and exploration of data. Hex supports SQL, Python, and no-code data manipulation, making it versatile for various data tasks.

## Hex + MotherDuck

Hex integrates with MotherDuck by allowing users to connect their Hex notebooks directly to MotherDuck's cloud data warehouse. This integration enables users to run SQL queries on MotherDuck's DuckDB-based backend, leveraging its high-performance and efficient data analytics capabilities within Hex's collaborative environment.
