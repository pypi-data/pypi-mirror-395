# Vasilije Markovic

Vasilije is currently building Cognee, an open-source framework for knowledge and memory management for LLMs. Before that, he has team lead at Taxfix where he managed and built their first data tools, worked at Zalando where he deployed machine learning applications, and Omio where he supervised the creation of real-time Big Data systems.
