# Meltano

INGESTION

Meltano is an open-source data integration platform designed for data teams to manage their ELT pipelines with ease. It supports a plug-and-play architecture, offering compatibility with various data sources and destinations, and allows for the orchestration of complex data workflows. Meltano emphasizes declarative code-first model, flexibility, and collaboration, making it a robust tool for streamlining data operations.

## Meltano + MotherDuck

Meltano integrates with MotherDuck by utilizing its ability to connect to DuckDB-based data warehouses. This seamless integration allows data teams to extract, load, and transform data directly within MotherDuck's scalable cloud environment, leveraging DuckDB's performance capabilities to optimize data workflows.
