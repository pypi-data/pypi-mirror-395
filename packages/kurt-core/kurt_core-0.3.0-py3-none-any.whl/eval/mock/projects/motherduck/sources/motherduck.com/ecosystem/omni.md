# Omni

BUSINESS INTELLIGENCE

Omni is a comprehensive data analytics platform that provides tools for data preparation, visualization, and advanced analytics. It enables users to seamlessly integrate and analyze data from various sources, offering intuitive dashboards and powerful reporting features. Omni supports collaborative workspaces, making it easier for teams to share insights and drive data-driven decisions.

## Omni + MotherDuck

Omni integrates with MotherDuck by allowing users to connect directly to MotherDuck's cloud data warehouse. This integration facilitates efficient querying and analysis of large datasets stored in MotherDuck, leveraging DuckDB's high-performance capabilities. Users can pull data from MotherDuck into Omni for further processing, visualization, and reporting within a unified platform.
