# Census

REVERSE ETL

Census is a reverse ETL tool that allows businesses to sync data from their data warehouse directly into the apps they use, such as CRMs, marketing platforms, and other SaaS tools. It helps ensure that all teams have access to the most up-to-date, actionable data by automating the data sync process. Census supports a wide range of data warehouses and enables data-driven workflows without requiring manual data exports or custom integrations.

## Census + MotherDuck

Census integrates with MotherDuck by using MotherDuck's cloud data warehouse capabilities built on DuckDB. This integration allows users to easily sync and transform data stored in MotherDuck to various business applications and SaaS tools. Consequently, data teams can leverage MotherDuck's robust querying and analytics features while ensuring that actionable insights are seamlessly propagated to operational tools via Census.
