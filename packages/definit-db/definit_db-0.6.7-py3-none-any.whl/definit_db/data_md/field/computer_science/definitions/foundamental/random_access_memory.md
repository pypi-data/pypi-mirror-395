# random access memory


[random access memory](computer_science/random access memory) is a type of [computer memory](computer_science/computer memory) that allows [data](computer_science/data) to be read or written 
in almost the same amount of time regardless of the physical location of the data within the memory. 
This enables fast access to any memory location, making it suitable for storing data that needs to be 
quickly accessed and modified during [program](computer_science/program) execution.
