# Features Collection

> **Stub** - This collection will be implemented in a future phase.

Features are behavioral modifiers that enhance agent capabilities like memory, RAG, or rate limiting.

## Planned Features

- `short-term-memory` - Recent conversation context
- `long-term-memory` - Persistent memory with summarization
- `knowledge-base` - RAG with custom retriever
- `humanizer` - Output humanization

## Usage

```bash
lk feature ls          # List available features
lk feature get <id>    # Clone feature to local-kit
lk feature set <id>    # Update feature configuration
```
